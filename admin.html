Conexión con Google Sheets

El inventario se sincroniza con tu Google Sheet publicado como CSV. Solo lectura para la tienda — escribe via Web App.

1

Abre tu Google Sheet → Archivo → Publicar en la web

2

Hoja Productos → formato CSV → Publicar → Copia la URL

3

Pega la URL abajo y haz clic en Conectar
